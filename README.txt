これはMyShogiで使える画像集です。

【使い方】
MyShogi.exeがあるフォルダにimageという名前でcloneしてください。

【編集について】
拡張子がidrawになっているファイルが元ファイルです。
Mac向けのGraphicというアプリで開けます。

【言い訳】
フリーのデータだけで「とりあえず」使える状態にすることを目標にしたものであり、
ちゃんとデザインされているわけではありません。
見栄えが悪いところも多々残っています。
(Pull Requestお待ちしております。)
